# VisionTalk Demo 视频草稿

**预计时长：3-5 分钟**

---

## 第一段：开场（15s）

```
（屏幕显示终端）
大家好，这是 VisionTalk——一个浏览器端的 AI 视觉对话助手。
打开浏览器，AI 就能看到你的画面、听懂你的声音。
不需要安装任何软件。
```

## 第二段：启动服务（20s）

```
（终端操作）
首先用 python backend/server.py 启动后端服务。
服务跑在本地 8765 端口。
打开浏览器访问 localhost:8765。
```

## 第三段：权限授权 + 摄像头（15s）

```
（浏览器画面：摄像头权限弹窗）
首次打开需要允许摄像头和麦克风权限。
点击允许后，左边出现摄像头实时画面。
右边是对话区域，红色按钮是 PTT 按键。
```

## 第四段：语音对话 — 核心演示（60s）

```
（点击 PTT 按钮，按钮变红）
点击红色按钮开始录音，对着麦克风说话。
「你好，介绍一下你自己是谁？」

（再次点击、等待）
再次点击结束录音。系统会依次显示：
- 🔊 语音转文字中...
- 转写结果：上方显示我说的话
- 🤖 AI 思考中...
- AI 流式回复，一个字一个字蹦出来

AI 介绍了自己是一个视觉助手。
底部显示耗时明细：STT 1.4 秒 + LLM 6 秒 = 7.4 秒。
```

## 第五段：视觉识别 — 重点演示（60s）

```
（举起手机对着摄像头）
现在演示视觉识别功能。

点击 PTT：「看看我手里面拿的是什么东西？」

（等待 AI 回复）
AI 正确识别出：你手里拿着一部智能手机，
手机背面有明显的摄像头模组和磁吸圆环。
你戴着眼镜，身穿黑色上衣。

说明 AI 确实"看到"了摄像头画面，不是瞎猜的。
```

## 第六段：再说一轮不同的物品（30s）

```
（换一个物品，比如杯子或书）
再试一个不同的东西。

点击 PTT：「现在我手里拿的又是什么？」
（等待回复）
AI 准确说出物品名称和特征。
```

## 第七段：查看日志（20s）

```
（打开 IDE 中的日志文件）
每次对话都有完整的日志记录，保存在 logs 目录下。
可以看到每一步的详细信息：
[1/3] 音频大小、解码耗时
[2/3] STT 转写结果和耗时
[3/3] LLM 输入文本、帧数、回复内容、首 token 延迟
```

## 第八段：技术架构简讲（20s）

```
（切回终端或截图）
技术架构很简单：
前端：原生 HTML/JS，浏览器采集摄像头和麦克风
后端：Python FastAPI + WebSocket
语音识别：阿里 DashScope Paraformer-v2 云端 API
大模型：通义千问 qwen3.7-plus 视觉模型
```

## 第九段：结尾（15s）

```
（回到浏览器画面）
VisionTalk 把视觉 AI 的门槛降到了最低——
打开浏览器就能用，不需要装任何软件。
欢迎大家 star 和试用，谢谢！
```
